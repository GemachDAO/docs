# 🚀 Getting Started

Welcome! If you're completely new to cryptocurrency and DeFi, you're in the right place.

{% hint style="info" %}
**You don't need to be a crypto expert to use GDex Pro.** GDex Pro is designed to be accessible to everyone — including people who have never owned or traded crypto before. This guide will give you the foundation you need to feel confident.
{% endhint %}

## What is this section for?

GDex Pro is a powerful decentralized trading platform — but we know that phrases like "decentralized," "wallet," "blockchain," and "DeFi" can sound intimidating if you've never encountered them before. This section explains all of those concepts in plain English, step by step, so you can go from zero to making your first trade with confidence.

## Good news: you don't need to understand everything to get started

GDex Pro supports **Web3Auth**, which lets you sign in with your existing **Google or Apple account** and creates a crypto wallet for you automatically. No confusing seed phrases, no browser extensions to install — just sign in and go. We'll explain how in the step-by-step walkthrough below.
