# 📟 GScanner

GScanner is available on Telegram.  The token scanner enables you to find the latest token deployments and snipe new trading pair creations before anyone else. Each time a new token is deployed a notification will be sent. Another notification message will be sent once the new token has trading enabled, so you can snipe it with GBot.\
